# Phase 08: State Tracker App — Scope

**Status**: planned

## Goal

Separate mobile app optimized for recording and resuming work/learning states during fragmented mobile access.

## Planned Deliverables

- React Native (Expo) app
- Quick state capture: what was I doing, what's next, where did I stop
- Resume flow: pick up any in-progress item with full context
- Syncs with Supabase (same backend as dashboard)

## Open Questions

- Should this be a separate app or a mode within the existing dashboard?
- How much overlap with the Next.js dashboard for task management?
- Offline-first or always-connected?
