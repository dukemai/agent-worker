# Phase 3: Dashboard UI — Scope

**Status**: done

## Goal

Build a web dashboard for viewing and managing tasks across the three time buckets.

## Deliverables

- Next.js app (App Router) with shadcn/ui components
- Three-column task layout (Today / This Week / Later)
- Mark done, move between buckets, add new tasks
- Source view to see original email bodies
- Context UI for editing family context key-value pairs
- Supabase Auth integration (login/logout, RLS)

## Out of Scope

- Mobile optimization (Phase 5)
- Renewal reminders (Phase 5)
- Daily digest (Phase 4)
