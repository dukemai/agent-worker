# Phase 13: Summer Activities - Scope

## Status

Active.

## Goal

Build a Summer Activities workflow for kids: collect Markdown suggestions, extract structured activities, separate reusable local activities from current-season instances, and surface practical options in the dashboard and daily digest.

## In Scope

- `/activities` dashboard route.
- Markdown source intake.
- AI extraction into reusable local activities and seasonal instances.
- Week panel for today, this week, rainy-day backups, booking-needed items, and evergreen fallbacks.
- Library and Seasonal review tabs.
- Daily digest section for timely activity suggestions.

## Out of Scope

- Booking activities directly.
- Full calendar sync.
- Public sharing.
- Perfect deduplication across all local venues.
- Web crawling or automatic source discovery.

## Acceptance Criteria

- A Markdown source can be saved and extracted.
- Extracted evergreen activities are stored separately from time-bound seasonal opportunities.
- The dashboard shows what can be done this week.
- The daily digest includes a concise Summer Activities section.
- Activities can be dismissed from the dashboard.
